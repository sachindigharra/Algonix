# Algonix

A DSA (Data Structures & Algorithms) practice and tracking platform — inspired by platforms like Codeforces,Leetcode,GFG but built with a smarter learning loop.

## What is Algonix?

Algonix helps you practice DSA problems while making sure you don't forget what you've already learned. As you progress through new topics, the app intelligently suggests previously covered topics for revision — keeping your knowledge fresh and retention high.

## Core Features

- **Problem Tracker** — Log and track DSA problems you've solved, with status (solved / attempted / pending)
- **Topic-wise Organization** — Problems grouped by topics (Arrays, Trees, Graphs, DP, etc.)
- **Pattern-based Categorization** — Problems tagged by solving patterns (e.g. Binary Search → Find First/Last, Search in Rotated Array, etc.) so you learn *how* to think, not just *what* to solve
- **Multi-dimensional Filtering** — Filter problems by Level (Easy / Medium / Hard), Company (Google, Amazon, etc.), Topic, and Pattern — independently or combined
- **Smart Revision Suggestions** — Based on your learning history, Algonix suggests older topics to revisit so you don't forget them over time
- **Progress Dashboard** — Visual overview of your solved problems, weak areas, and revision streaks
- **Notes & Hints** — Attach personal notes or hints to any problem for future reference
- **Platform Aggregator** — Track your progress across multiple coding platforms (LeetCode, Codeforces, GFG, HackerRank, etc.) all in one place — no more switching tabs to check your stats
